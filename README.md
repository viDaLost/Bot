# Sunset Telegram Bot Pro

Telegram-бот для автоматической публикации красивой картинки с временем «заход солнца минус 1 час».

## Возможности

- Управление через inline-кнопки.
- Каналы и чаты: добавление, список, проверка доступа, удаление.
- Расписания:
  - один раз;
  - каждый день;
  - по выбранным дням недели;
  - каждый месяц.
- Предпросмотр картинки перед сохранением.
- Тестовая отправка в канал/чат.
- Пауза, возобновление и удаление расписаний.
- Изменение времени, стиля и заголовка расписания.
- Несколько стилей картинки.
- Русская дата на картинке.
- История отправок.
- Уведомления админу об ошибках отправки.
- Экспорт и импорт базы bot.db.
- В канал/чат отправляется только картинка без подписи.

## Файлы

- `bot.py` — меню, кнопки, сценарии создания и управления.
- `scheduler.py` — планировщик отправок.
- `database.py` — база SQLite и миграции.
- `image_generator.py` — генерация картинок.
- `config.py` — чтение `.env`.
- `requirements.txt` — зависимости.

## Запуск на Wispbyte

Startup command:

```bash
pip install --user -r requirements.txt && python bot.py
```

App py file:

```text
bot.py
```

Создай файл `.env` в Files:

```env
BOT_TOKEN=НОВЫЙ_ТОКЕН_БОТА
ADMIN_IDS=1288379477
DATABASE_PATH=bot.db
```

Не загружай `.env` в GitHub.

## Важно

В этой версии нет режима автоматической отправки ровно за 1 час до фактического заката. Бот отправляет публикацию в выбранное тобой время, а на картинке показывает рассчитанное время «заход солнца минус 1 час».
